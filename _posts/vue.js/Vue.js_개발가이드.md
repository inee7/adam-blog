# Vue.js 개발 가이드

## vue project 생성

*brew, git, npm 설치 필수

공식 지원 구성인 vue-cli로 프로젝트를 생성한다.

### vue-cli 설치

```bash
npm install -g vue-cli
```

### vue프로젝트 생성

```bash
vue init webpack my-project
```



### vue 실행

```bash
  cd front-end
  npm run dev
```



### vue production code 생성

```bash
npm run build
```
